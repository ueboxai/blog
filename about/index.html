About the project

Tools should reduce uncertainty.

Unreal Box is a Windows desktop workspace for Unreal Engine creators. It is designed to make projects, source assets, technical context and repeatable workflows easier to find and inspect.

What the beta is for

The current beta explores a practical layer around Unreal Editor: project and engine discovery, reference and backup asset libraries, previews, metadata, optional editor context, technical knowledge and visual workflows.

Unreal Box does not replace Unreal Editor, Perforce, Git or a team's review process. It should make the decisions around those systems more visible.

How we write

  • Prefer steps a developer can test over abstract claims.
  • Separate verified behavior from roadmap ideas.
  • Keep destructive actions previewable and reversible.
  • Do not publish user counts, efficiency percentages or testimonials without evidence and permission.
  • Invite criticism from developers working on real projects.
