## 技术栈

- Hugo 静态站点生成器
- GitHub Pages 托管
- 自动化内容发布

## 本地开发

```bash
# 安装 Hugo
brew install hugo

# 克隆仓库
git clone https://github.com/uebox/blog.git
cd blog

# 启动开发服务器
hugo server -D
```
